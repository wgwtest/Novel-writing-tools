# 叙事验证工具 - 人物条实虚段原型 v8

## 版本目的

本版用于验证“事件线”和“人物条”的职责边界：

- 事件线只显示正文中实际发生、实际叙述到的事件段。
- 人物条是某个人物在某条事件线下的参与视图。
- 人物条实框表示正文已经写到的人物行为。
- 人物条虚框表示逻辑存在但正文未直接写出的后台状态。

## 相对 v7 的变化

- 新建 v8 原型包，保留 v7 作为已提交基准。
- 将“禹树 / 大仙祠线”的长人物条拆成：
  - `情报入口显露`：正文可见实段。
  - `情报后台发酵`：后台虚段。
- 事件线块不再承担人物后台延续含义。
- Inspector 会区分 `EXPLICIT` 和 `BACKGROUND`。
- 选中后台虚段时，右侧提示它不能替代正文事件，也不能扩大故事线边界。

## 验收重点

1. 故事线事件块只表达正文可见事件。
2. 人物条可以跨越故事线事件块之外，但超出的部分必须使用虚框。
3. 虚框必须被 Inspector 解释为后台状态，而不是新的正文事件。
4. 同一人物仍可出现在多条故事线下，但每条线下的片段语义独立。
