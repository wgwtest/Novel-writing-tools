- 版本：v9

- 目标画板：1920 x 1080

## 本版定位

V9 只验证一个核心判断：故事由事件组成，但事件必须落在真实时间底盘上；章节不是横轴本体，而是覆盖在真实时间上的正文切片。

本版默认时间范围为：

```text
2025-07-07 08:00 -> 2025-08-01 00:00
```

这对应“2025 年 7 月第二周开始，到 7 月底结束”的演示范围。该时间只是原型默认值，不代表小说最终时间表已经确认。

## 非目标

- 不做复杂日历编辑器。
- 不做拖拽修改日期。
- 不做自动保存。
- 不修订正式小说时间线。
- 不把演示日期写成最终设定。

## 设计依据

- 用户要求：时间轴应能对应真实时间，并且起止时间可设置。
- 用户补充：第二卷可从 2025 年 7 月第二周附近开始，先选一个时间作为起止范围。
- v8 基准：事件线只显示正文可见事件；人物条区分实段和后台虚段。
- 设计判断：真实时间轴应成为底盘，章节覆盖层只是写作切片。

## 核心设计

### 1. 真实时间轴

画布顶部第一层显示真实日期刻度。默认从 `2025-07-07 08:00` 到 `2025-08-01 00:00`。

工具栏提供起止时间输入框，点击“应用”后重新计算画布宽度和所有对象位置。

日期刻度只作为顶部标尺，不贯穿主画布。

### 2. 章节覆盖层

章节不再等宽。每章根据 `startAt / endAt` 显示为真实时间轴上的覆盖区。

点击章节只选中章节覆盖区，不自动进入章节函数。

主画布的纵向网格以章节覆盖边界为准，而不是以每天为网格单位。

### 3. 事件线与人物条

事件、人物实段、人物虚段都使用真实时间字段：

```text
startAt
endAt
```

事件线继续只表示正文可见事件。人物条继续区分：

- 实框：正文写到的人物行为。
- 虚框：后台存在但未直接写出的人物状态。

虚框必须是完整虚线框，不再绘制实段使用的底部强调线。

### 4. Inspector

选中事件或人物条后，Inspector 显示：

- 真实时间范围。
- 覆盖章节。
- 所属故事线。
- 人物条的 `EXPLICIT / BACKGROUND` 状态。

## 原型到实现映射

- 页面归属：叙事验证工具 / 多轨时间轴工作台。
- 主对象：真实时间轴、章节覆盖、事件线、人物条。
- 数据来源：当前原型内置演示数据。
- 验收方法：打开 `source/index.html`，检查默认视图、时间范围设置、选中态 Inspector。

## 允许偏差

- 示例日期可调整。
- 章节覆盖时间可根据正式时间线重设。
- 日期刻度密度可根据缩放比例继续细化。
- 顶部日期标尺样式可调整，但不得变成主画布网格。

## 不可接受偏差

- 不能回退为等宽章节轴。
- 不能用章节编号直接决定事件块位置。
- 不能让日期刻度贯穿主画布并替代章节网格。
- 不能让人物后台虚段扩展故事线事件边界。
- 不能给人物虚段额外叠加实段式底部强调线。
- 不能把演示时间误写成正式设定。
