# v25 评审响应

## 来源意见

用户提出：至少要打通大时间轴和章节工作台的联系。具体来说，从大时间轴里选中一个章节进入时，要能看到该章节的时间线工作台。

## 处理结论

本版接受该意见，并将其收敛为一条明确路径：

```text
全局时间画布选中章节 -> Inspector 确认 -> 进入章节工作台
```

## 实现调整

1. 新增全局时间画布入口。
2. 支持点击章节列选中章节。
3. 选中章节后右侧 Inspector 显示章节对象信息。
4. Inspector 中提供 `进入章节工作台` 按钮。
5. 点击按钮后进入 `#chapter/<chapterId>`。
6. 章节工作台根据章节 ID 渲染对应章节的输入、延续、生成和输出事件。
7. 提供返回全局时间轴按钮。

## 本版不处理

- 不展开完整事件详情页。
- 不实现持久化。
- 不实现复杂拖拽。
- 不处理多章节批量编辑。
