# Character Avatar Prompts

Mode: built-in `image_gen`

Use: square semi-realistic cartoon avatars for the narrative analysis tool.

Shared style:

- 1:1 square character avatar.
- Head-and-shoulders to upper-chest framing.
- Semi-realistic cartoon / graphic-novel portrait.
- Realistic proportions, clean expressive linework, soft painterly shading.
- Low-saturation cinematic color.
- One person only.
- No text, no Chinese characters, no letters, no watermark, no logo.
